Remarque : Après un essai gratuit de trois jours, l'application doit être déverrouillée via un achat unique.

Cette application open-source utilise les capacités HDR de votre MacBook Pro pour augmenter la luminosité maximale de votre écran pour n'importe quelle application !
L'application propose divers paramètres tels que les niveaux de luminosité, les raccourcis et les automatisations pour rendre votre expérience aussi pratique que possible.

Les appareils pris en charge sont :

- MacBook Pro M5 14" de 2025 (Identifiant de modèle : Mac17,2)
- MacBook Pro M4 de 2024 (Identifiants de modèle : Mac16,1, Mac16,5, Mac16,6, Mac16,7, Mac16,8)
- MacBook Pro M3 de 2023 (Identifiants de modèle : Mac15,3, Mac15,6, Mac15,7, Mac15,8, Mac15,9, Mac15,10, Mac15,11)
- MacBook Pro M2 14" / 16" de 2023 (Identifiants de modèle : Mac14,5, Mac14,6, Mac14,9, Mac14,10)
- MacBook Pro M1 14" / 16" de 2021 (Identifiants de modèle : MacBookPro18,1, MacBookPro18,2, MacBookPro18,3, MacBookPro18,4)
- Pro Display XDR

De bons indicateurs sont les deux questions suivantes : L'écran de votre MacBook a-t-il une encoche, et est-ce qu'il est inscrit MacBook Pro au dos de votre appareil ?

Caractéristiques :
- Contrôlez votre luminosité maximale d'écran avec un menu pratique dans la barre de menus
- Raccourcis clavier
- Ouvrez les paramètres via la recherche Spotlight en cherchant "BrightIntosh Settings" ou un raccourci clavier (Par défaut : Cmd+Option+Shift+B)
